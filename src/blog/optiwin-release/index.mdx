## 项目简介

OptiWin 是一款适用于 Windows 系统的个性化调整工具箱，基于 [meetrevision/revision-tool](https://github.com/meetrevision/revision-tool) 二次开发。采用 Go + Wails 后端与 Vue 3 + Naive UI 前端技术栈，提供现代化的界面和便捷的系统优化体验。

## 功能模块

### 🏠 首页
- 项目信息展示
- 系统信息实时显示（OS / CPU / 内存 / IP）

### 🔒 安全
- Windows Defender 引擎禁用/服务禁用/恢复
- UAC 设置管理
- VBS（虚拟化安全）开关
- 内存完整性控制

### ⚡ 性能
- 电源计划优化
- C-State 控制
- Superfetch 管理
- 内存压缩开关
- 全屏优化设置
- 窗口优化调整
- MPO（多平面优化）控制
- 着色器缓存管理
- Xbox 服务（Game Bar）独立控制

### 🎨 个性化
- 通知系统管理
- 气球通知开关
- 边缘滑动设置
- 上下文菜单自定义
- Explorer 主页和图库配置
- 快捷方式外观调整
- Win11 新版任务管理器开关

### 🛠️ 实用工具
- 休眠功能管理
- 快速启动开关
- 照片查看器恢复
- Edge 浏览器卸载
- WebView2 管理
- 安全模式启动
- 进入 BIOS 快捷方式

### 🔄 更新
- 证书更新管理
- KGL 更新控制
- 暂停更新功能
- 隐藏更新页面
- 驱动更新策略
- 更新通道切换
